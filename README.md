# Calculating-the-sum-of-two-numbers
Скласти схему алгоритму і написати програму мовою Phyton, розв’язання завдань яких організувати у функціях. Тобто основна програма повинна містити введення вхідних даних, виклик функції, в якій реалізовано розв’язання, та виведення результатів

# Початок коду
def sum_numbers(a, b):
    return a + b 
# Назва що робить програма
print("Програма для обчислення суми двох чисел")
# Дані(обчислення)
a = float(input("Введіть перше число: "))
b = float(input("Введіть друге число: "))

result = sum_numbers(a, b)
# Результат
print("Сума чисел: =", result)
